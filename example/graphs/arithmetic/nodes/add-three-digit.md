# Introduction

The column method extends unchanged to hundreds: the same carry rule just
runs one column further. This lesson exists mostly to consolidate carrying.

## KP 1: Three-Digit Sums Without Carrying

### Worked Example

312 + 456: ones 2 + 6 = 8, tens 1 + 5 = 6, hundreds 3 + 4 = 7 -> 768.

### Problems

1. 234 + 145
2. 502 + 306
3. 421 + 268

## KP 2: Three-Digit Sums With Repeated Carrying

### Worked Example

387 + 245: ones 7 + 5 = 12 -> write 2, carry 1. Tens 8 + 4 + 1 = 13 ->
write 3, carry 1. Hundreds 3 + 2 + 1 = 6. Result: 632.

### Problems

1. 276 + 348
2. 489 + 233
3. 555 + 367
